## Input tag의 속성

* autocomplete = on / off
	* 자동 완성기능이 활성화됨(비활성화됨)
* placeholder = "value"
	* text에 value값이 기본값으로 나타나게됨 
* autofocus
	* 사용자의 커서가 이 옵션이 있는 곳으로 감
